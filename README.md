# Halo 滚动数字组件

为 Halo 2.25.x 默认编辑器增加一个可视化的滚动数字内容组件。组件第一次进入浏览器可见区域时播放一次，离开后再次进入不会重复播放；数字滚动，普通文字和符号保持静止。

## 功能

- 整数：`1000`
- 小数：`100.90`
- 百分比：`98.6%`
- 前缀与后缀：`¥100.90万+`
- 英文日期：`2016y12m21d`
- 中文日期：`2016年12月11日`
- 一段内容内可包含多组数字，每组数字同时滚动
- 默认动画总时长为 1.5 秒
- 自定义内容、字号、颜色、字重、对齐、宽度和动画时长
- 宽度支持 `100%`、`auto`、`480px` 等 CSS 宽度写法，并限制为不超过容器宽度
- 尊重系统“减少动画”设置

## 安装

1. 下载 Release 中的 JAR 文件。
2. 进入 Halo 后台的“插件”，点击右上角“安装插件”。
3. 上传 JAR，安装并启用。
4. 建议启用后清理主题缓存或强制刷新浏览器一次。

## 使用

1. 打开 Halo 默认文章编辑器。
2. 在编辑器工具箱中选择“滚动数字”，也可以输入 `/滚动数字` 插入。
3. 点击已插入的组件，在下方调整各项参数。
4. 保存并发布。访客向下滚动、组件首次进入可见区域时，动画自动播放一次。

## 开发构建

需要 JDK 21、Node.js 和 pnpm。

```bash
./gradlew clean build
```

构建完成的插件位于 `build/libs/`。

## 兼容性

- Halo：`>= 2.25.0`
- 编辑器：Halo 默认编辑器
